# AGENTS.md

本项目是中文 F1 新手知识库，目标是用静态内容帮助读者理解比赛规则、策略、车手操作、赛道弯角和学习资源。

## 维护规则

1. 不要使用官方 F1 logo、车队 logo、官方转播截图或未经授权图片。
2. 不要抓取官方网站或大规模复制官方数据。
3. 新增内容要保持中立、解释性、面向新手。
4. 规则类内容要注明可能随赛季变化，并提醒读者以 FIA 和 Formula 1 官方发布为准。
5. 页面优先使用 Markdown 和已有 Vue 组件，不要引入大型 UI 框架。
6. 不要添加后端、数据库、登录系统、评论系统、付费系统或复杂状态管理。
7. 新增外部资源链接时，要检查链接有效性，并说明资源适合的人群和使用场景。
8. 修改后必须运行：

```bash
npm run docs:build
```

## 内容模板

统一模板位于 `docs/templates/`：

- `rule-template.md`
- `strategy-template.md`
- `racecraft-template.md`
- `track-template.md`

新增页面时优先复制对应模板，并链接到相关概念页面和术语表。

## 写作风格

- 清楚、克制、准确。
- 避免饭圈化和绝对化判断。
- 解释策略时优先使用时间差、轮胎状态、交通和赛道位置。
- 对不确定或会变化的内容使用“通常”“可能”“取决于赛道/轮胎/规则”等表达。
