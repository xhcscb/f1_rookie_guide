# 关于本站

赛车战术板 / Racecraft 中文手册是一个面向中文 F1 新手的静态知识库。它的目标是帮助读者建立观赛框架：先看懂比赛周末、规则、策略、车手操作和赛道弯角，再逐步接触更复杂的数据和技术内容。

## 当前阶段

第一版聚焦静态内容和信息架构，不提供实时数据、不做预测模型、不抓取官方网站或转播素材，也不使用 F1 官方 logo 或车队 logo。后续可以扩展自绘赛道图、分站观赛卡、数据复盘和术语搜索。

## 维护原则

内容应保持中立、解释性和面向新手。涉及规则、赛历、处罚和车队车手信息时，需要提醒读者这些内容可能随赛季变化，并尽量链接到官方或可信来源。
