# 术语表

术语表按主题组织，每个词先给 1-2 句观赛解释，再链接到相关页面。遇到直播术语时，可以先在这里快速定位。

## 规则与比赛流程

| 术语 | 解释 | 相关页面 |
| --- | --- | --- |
| DRS | 可变尾翼减阻系统，后车通常需要在检测点进入前车 1 秒内，才能在指定区域打开尾翼降低阻力。 | [DRS 与 Overtake Mode](/rules/drs-overtake-mode) |
| Overtake Mode | 常指车手使用电能部署增强直道加速的模式，与 DRS 不是同一件事。 | [DRS 与 Overtake Mode](/rules/drs-overtake-mode) |
| track limits | 赛道限制，通常围绕赛车是否越过规定边界并从中获利。 | [赛道限制](/rules/track-limits) |
| safety car | 安全车会压缩车距并带领全场慢速行驶，常改变进站成本和比赛节奏。 | [安全车与 VSC](/rules/safety-car-vsc) |
| virtual safety car | 虚拟安全车通过限速控制全场节奏，不像实体安全车那样把车队压成同一列。 | [安全车与 VSC](/rules/safety-car-vsc) |
| parc ferme | 排位后赛车进入受限调整状态，车队不能随意更改关键设定。 | [排位赛](/guide/qualifying) |

## 策略与轮胎

| 术语 | 解释 | 相关页面 |
| --- | --- | --- |
| undercut | 先于对手进站换新胎，用新胎速度追回时间，争取对手进站后出在其前方。 | [Undercut](/strategy/undercut) |
| overcut | 比对手晚进站，利用旧胎仍有速度、清洁空气或对手交通损失来反超。 | [Overcut](/strategy/overcut) |
| pit window | 进站窗口，指进站后能落在理想位置的时间范围。 | [进站窗口](/strategy/pit-window) |
| pit loss | 完成一次进站相对不停站损失的总时间，包含进出维修区和换胎。 | [进站与 pit loss](/rules/pit-stop) |
| stint | 一套轮胎连续完成的一段比赛。进站后通常开始新的 stint。 | [轮胎衰退](/strategy/tyre-degradation) |
| compound | 轮胎配方，例如软、中、硬胎；不同配方在速度和寿命上取舍不同。 | [轮胎规则](/rules/tyres) |
| degradation | 轮胎衰退，指轮胎随使用逐渐失去速度或稳定性。 | [轮胎衰退](/strategy/tyre-degradation) |
| graining | 胎面橡胶颗粒化，可能导致抓地下降，常与温度窗口和滑动有关。 | [轮胎规则](/rules/tyres) |
| blistering | 胎面起泡，通常与过热和负荷有关，会影响抓地和轮胎寿命。 | [轮胎规则](/rules/tyres) |
| DRS train | 一串赛车都在前车 1 秒内，大家都有 DRS，导致后车难以进一步超车。 | [DRS Train](/strategy/drs-train) |

## 车手操作

| 术语 | 解释 | 相关页面 |
| --- | --- | --- |
| dirty air | 前车扰乱后的气流，后车跟近时下压力下降，高速弯和轮胎温度尤其受影响。 | [脏空气](/strategy/dirty-air) |
| slipstream | 尾流，后车在直道跟随前车时可能减少空气阻力，提高尾速。 | [DRS 与 Overtake Mode](/rules/drs-overtake-mode) |
| apex | 弯心，赛车通过弯角时最靠近内侧的位置，影响出弯方向和速度。 | [弯心](/racecraft/apex) |
| racing line | 赛车线，车手选择的过弯路径，目标是平衡入弯、弯中和出弯速度。 | [赛车线](/racecraft/racing-line) |
| lock-up | 锁死，刹车时轮胎停止有效滚动并滑动，可能导致刹车距离变长和轮胎受损。 | [刹车点](/racecraft/braking-point) |
| flat spot | 锁死后轮胎被磨出平点，严重时会带来振动并迫使提前进站。 | [刹车点](/racecraft/braking-point) |
