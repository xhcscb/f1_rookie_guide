# 新手路线

第一次看 F1 不需要马上理解全部技术词汇。更有效的方式是先建立观看顺序：比赛周末如何展开、排位决定什么、正赛中哪些时刻会改变局面，然后再补轮胎、进站和 DRS 等概念。

## 建议阅读顺序

1. 先读[第一次看 F1](/guide/watching-f1)，知道直播中优先关注什么。
2. 再读[比赛周末](/guide/weekend-format)，理解每节比赛的目标不同。
3. 排位前读[排位赛怎么看](/guide/qualifying)，正赛前读[正赛怎么看](/guide/race)。
4. 遇到术语时回到[术语表](/glossary/)查 1-2 句解释。
