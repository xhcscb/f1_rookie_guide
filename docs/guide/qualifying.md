# 排位赛怎么看？

## Q1/Q2/Q3 是什么

排位赛通常分为 Q1、Q2、Q3 三段淘汰。Q1 先淘汰最慢的一批车手，Q2 再淘汰一批，Q3 决定最前方发车顺位。每位车手要在有限时间里找到干净空间、把轮胎带进工作温度，并在关键圈尽量少犯错。

## 为什么排位赛很重要

排位决定正赛起步位置。发车靠前通常意味着空气更干净、交通更少、策略选择更主动。难超车赛道尤其重视排位，因为正赛里即使后车稍快，也可能长期被困在前车后方。

## Tow、track evolution、traffic

**Tow** 指前车尾流帮助后车在直道获得更高尾速，常见于蒙扎等高速赛道。**Track evolution** 指赛道随着橡胶附着和温度变化逐渐变快，所以最后几分钟经常有人刷新成绩。**Traffic** 是交通：慢车或准备圈车辆挡住快圈节奏，会让车手损失时间。

## 为什么队友对比很关键

同队赛车通常拥有相近基础性能。排位中队友差距能帮助你判断：这是赛车整体速度不足，还是某位车手在某些弯、轮胎准备或压力下表现更好。单站差距可能受事故、交通影响，长期趋势更有参考价值。
