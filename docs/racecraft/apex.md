# 弯心

## 一句话解释

弯心是赛车在通过弯角时最靠近内侧的位置，它影响转向节奏和出弯方向。

## 观赛时怎么看？

看车手是否在合适位置贴近内侧路肩。有些弯需要早弯心，快速把车带向出口；有些弯需要晚弯心，牺牲入弯速度换更好出弯，尤其是后面接长直道时。

## 为什么它影响超车或圈速？

弯心决定车身能否尽快回正。错过弯心可能导致出弯角度差、加油晚，直道末端就会失去防守或进攻机会。排位中一个小错过弯心的动作，可能累积成明显时间损失。

## 常见误解

贴内侧不一定等于过弯好。某些路肩太高或太滑，压上去会让赛车跳动；有些复合弯需要为了下一弯牺牲当前弯心。不同弯角的“正确弯心”取决于前后组合。

## 和其他概念的关系

弯心连接赛车线和出弯速度。防守时守内线可能让对手难以切入，但也可能让自己弯心过早、出口变慢，被对手交叉线反击。
