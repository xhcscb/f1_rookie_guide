# 刹车点

## 一句话解释

刹车点是车手开始明显减速的位置，决定入弯速度、能否守住线路和是否有机会在刹车区超车。

## 观赛时怎么看？

重刹车弯前，后车如果借尾流和 DRS 接近，可能选择更晚刹车攻击内线。你可以看两车是否并排进入刹车区、谁先锁死、谁能在弯心前控制住赛车。

## 为什么它影响超车或圈速？

晚刹车能带来进攻机会，但也更容易错过弯心或损失出弯。排位中车手追求极限刹车点；正赛中，如果轮胎已经衰退，过晚刹车可能导致锁死、flat spot 或被对手反击。

## 常见误解

更晚刹车不一定更快。刹得太晚可能入弯过深，出弯被迫慢下来。很多成功超车不是“谁最晚刹”，而是谁能在刹车、转向和出弯之间保持整体速度。

## 和其他概念的关系

刹车点与赛车线、弯心、轮胎抓地和防守路线紧密相关。进攻方需要在刹车区占到足够位置，防守方则可能提前选择内线，让对手走更差角度。
