# 出弯速度

## 一句话解释

出弯速度是赛车离开弯角进入直道时的速度和加速质量，很多直道超车的基础其实发生在这里。

## 观赛时怎么看？

不要只看直道末端。先看直道前一个弯：后车是否更早加油、车身是否更稳定、有没有借外侧获得更大转弯半径。若后车出弯已经贴近，直道上的尾流、DRS 和能量部署才更容易发挥。

## 为什么它影响超车或圈速？

长直道前的出弯速度会被直道长度放大。出弯快 0.1 秒，到了刹车点可能转化成更明显的并排机会。反过来，入弯过猛导致出弯慢，哪怕刹车点看起来很勇，也可能整段亏时间。

## 常见误解

直道超车不是只靠马力或 DRS。很多时候，后车能不能进入 DRS 攻击距离，取决于前一个弯有没有把赛车摆正、有没有保护好后胎、有没有避免打滑。

## 和其他概念的关系

出弯速度与弯心、赛车线、轮胎状态和 DRS Train 都有关。难超车赛道上，车手常常连续几圈准备同一个出弯，只为在下一条直道获得一次干净攻击。
