# 中文内容

第一版暂不直接列出中文创作者推荐。中文 F1 内容生态变化快，且质量差异较大，贸然推荐容易过时。这里先给出筛选标准，后续可按标准逐步补充。

## 推荐筛选标准

1. 长期更新，而不是只在爆点事件后输出。
2. 引用来源清楚，能区分官方消息、媒体报道和个人观点。
3. 解释概念时面向新手，但不把复杂问题简化成情绪标签。
4. 讨论车手和车队时保持中立，不靠攻击性表达制造流量。
5. 对规则、处罚和技术细节保留不确定性，愿意更正错误。
