# DRS 与 Overtake Mode 入门

## 一句话解释

DRS 是可变尾翼减阻系统，Overtake Mode 通常指车手使用电能部署的进攻模式；一个改变空气阻力，一个改变能量输出。

## 比赛中怎么看出来？

DRS 只能在指定检测点和激活区使用，通常要求后车在检测点距离前车 1 秒以内。镜头上可以看到尾翼打开，直道尾速提升。Overtake Mode 不一定有明显外观变化，更多通过车手无线电、方向盘操作或直道加速效果体现。

## 为什么它重要？

现代 F1 直道攻防经常依赖 DRS、尾流和能量部署的叠加。后车即使有 DRS，也需要前一个弯出弯足够近；前车则可能保留电能，在直道上防守。两者让超车变成多圈铺垫，而不是单一按钮。

## 新手常见误解

DRS 不是“自动超车键”。如果后车出弯差、轮胎过热、前车也有尾流或赛道直道太短，DRS 仍可能不够。Overtake Mode 也受电池能量、部署策略和规则限制，不可能无限使用。
