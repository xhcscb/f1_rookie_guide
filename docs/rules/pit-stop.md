# 进站与 pit loss

## 一句话解释

Pit loss 是赛车完成一次进站相对不停站所损失的总时间，包含减速进维修区、换胎、限速行驶和回到赛道的时间。

## 比赛中怎么看出来？

当车队讨论“窗口”时，通常是在计算进站后会落到谁前后。如果某条赛道 pit loss 约二十多秒，车手领先身后对手不到这个差距，进站后很可能掉到后面。安全车或虚拟安全车期间，pit loss 往往会变小。

## 为什么它重要？

进站是策略的核心开关。换新胎能换来速度，但必须先付出 pit loss。一次成功进站不只看 2 秒多的换胎动作，还要看进站时机、出站交通、轮胎升温速度和与对手的相对位置。

## 新手常见误解

“换胎快”不等于策略成功。维修区动作快但出站被慢车挡住，仍可能亏时间；换胎慢半秒但出站空气干净，也可能总体更好。Pit loss 也不是固定常数，会受安全车、维修区长度和赛道入口出口布局影响。
