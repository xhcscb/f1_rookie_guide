# 赛道限制

## 一句话解释

赛道限制通常指赛车是否越过规定赛道边界，常见判断基准是白线；违反后可能删圈、警告或罚时。

## 比赛中怎么看出来？

排位赛中，如果车手四轮超出赛道边界，快圈可能被删除。正赛中，多次违反可能先被警告，之后受到罚时。直播常在弯角出口提到 track limits，因为车手为追求出弯速度会把赛车推到赛道边缘。

## 为什么它重要？

赛道限制影响公平性。多利用赛道外空间，可能获得更高出弯速度、更小转向角或更早全油门。如果不限制，车手会不断扩大可用路线，比赛变得难以比较。

## 新手常见误解

“看起来只是压线”不一定安全，也不一定违规，关键在具体规则和赛事说明。不同赛道的高风险弯角不同，某些分站会因为摄像头、传感器或赛道边界设计而频繁出现删圈。
