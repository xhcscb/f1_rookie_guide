# 轮胎规则与软 / 中 / 硬胎

## 一句话解释

F1 轮胎配方决定速度、寿命和温度窗口；软胎通常更快但更不耐用，硬胎通常更耐用但不一定容易立刻发挥速度。

## 比赛中怎么看出来？

直播画面通常会显示每位车手的轮胎颜色、已跑圈数和进站次数。你可以观察：某位车手是否在旧胎上圈速掉得明显，是否开始频繁被后车接近，或者无线电里提到 graining、overheating 等问题。

## 为什么它重要？

轮胎直接决定策略空间。早停换新胎可能获得短期速度，但会让后段轮胎更老；晚停能保留新胎冲刺，但前段可能被困在慢车后方。不同赛道对轮胎的横向、纵向负荷也不同，所以同一种配方在不同分站表现会差很多。

## 新手常见误解

软胎不等于永远最快，硬胎也不等于永远最慢。轮胎需要温度窗口，过冷会没有抓地，过热会衰退。规则细节和可用配方会随赛季、分站变化，新手先理解“速度与寿命交换”即可。
