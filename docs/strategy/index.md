# 策略

F1 策略不是玄学，也不是一句“车队算得准”。它通常是时间差、轮胎状态、交通、赛道位置和安全车概率的组合。新手先学会问四个问题：现在换胎能赚多少速度？进站会掉到哪里？前后有没有慢车？后段轮胎还能不能撑住？

## 阅读方法

每个策略页面都按同一结构解释：一句话、解决什么问题、比赛中怎么看、成功条件、失败原因、简化例子和相关页面。遇到直播里的策略判断时，先把它还原成“时间”和“位置”，再讨论车队选择是否合理。
