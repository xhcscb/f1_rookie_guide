# 看懂车队

车队不是普通俱乐部，而是赛车研发、策略执行、赛事运营和车手表现的综合体。新手看车队时，不必急着记住每个部门或赞助名，先抓四条线索：赛车特性、策略风格、升级节奏和队友对比。

## 车队表现从哪里看？

**赛车特性**：有的车低速弯好，有的车高速稳定，有的车直道效率强。单站成绩要结合赛道特性看，不能只用一场比赛判断全年水平。

**策略风格**：车队是否保守覆盖，是否愿意分拆策略，安全车窗口反应是否快，都会改变结果。

**升级节奏**：F1 是研发竞赛。赛季中带来的底板、前翼、冷却和减重更新，可能改变车队相对位置。

**队友对比**：同队两位车手的差距，是判断赛车和车手的关键参照。若两人都慢，可能是赛车问题；若差距长期稳定，才更值得讨论车手差异。
