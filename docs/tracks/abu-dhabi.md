# 亚斯码头赛道

<TrackMap track="abu-dhabi" />

## 一句话特点

长直道加低速段。策略和刹车区攻防很清楚。

## 新手先看什么？

看两条长直道后的刹车区。DRS 和防守线会反复出现。
