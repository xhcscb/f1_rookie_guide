# 巴林国际赛道

<TrackMap track="bahrain" />

## 一句话特点

重刹、牵引、后胎管理。看 T1 和 T10，就能抓住大半节奏。

## 新手先看什么？

看赛车出慢弯后能不能稳住后胎。后胎一滑，直道速度和防守都会掉。
