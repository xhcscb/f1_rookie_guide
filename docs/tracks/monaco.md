# 摩纳哥

## 一句话特点

摩纳哥是窄街道、低速弯和护墙压力的代表，排位极其重要，正赛超车非常困难。

## 新手先看什么？

先看排位。摩纳哥的赛道宽度和低速弯让正赛很难靠纯速度完成超车，因此单圈位置价值很高。正赛中则重点看发车、进站窗口、安全车和车手是否能在护墙之间保持稳定。

## 为什么这条赛道容易 / 不容易超车？

摩纳哥不容易超车，因为直道短、赛道窄、弯角密集，后车很难获得足够并排空间。即使后车轮胎更新，也可能长期被前车压住，策略往往比赛道超车更重要。

## 排位赛看点

排位看点是交通管理、轮胎准备和车手贴墙精度。一次 traffic 或小失误就可能毁掉快圈，因此车队放车时机非常关键。

## 正赛看点

正赛重点看前排能否守住、进站是否被 undercut/overcut 改变，以及安全车是否让某些车手获得便宜进站。长时间跟车时，后车更可能等待策略窗口，而不是硬超。
