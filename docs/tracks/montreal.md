# 吉尔·维伦纽夫赛道

<TrackMap track="montreal" />

## 一句话特点

重刹、低速出口、墙。很容易出安全车。

## 新手先看什么？

看刹车稳定和出弯牵引。这里不是弯多，而是每次刹车都要准。
