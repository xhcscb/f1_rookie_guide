# 蒙扎

## 一句话特点

蒙扎是高速、低下压力和重刹车的代表，尾流、DRS 和刹车区执行非常关键。

## 新手先看什么？

先看直道末端的减速弯，再看车手如何处理尾流。蒙扎不是只有“谁直道最快”，因为低下压力设定会让刹车稳定性和出弯牵引也变得敏感。

## 为什么这条赛道容易 / 不容易超车？

蒙扎相对适合超车，因为长直道和重刹车提供并排机会。但如果形成 DRS Train，前车也有 DRS，后车优势会被抵消。排位中的 tow 也会显著影响单圈成绩。

## 排位赛看点

排位看 tow 和 traffic。车手想借前车尾流，但又不能离得太近影响弯中下压力。车队放车时机和队友配合经常成为蒙扎排位焦点。

## 正赛看点

正赛重点看 DRS Train、刹车区攻防和低下压力设定下的轮胎保护。直道速度快的车不一定正赛最舒服，因为防守和刹车稳定同样重要。
