# 斯帕

## 一句话特点

斯帕是一条长圈、海拔变化和高速组合明显的经典赛道，天气、尾流和下压力取舍都会影响比赛。

## 新手先看什么？

先看 Eau Rouge/Raidillon 到 Kemmel 直道，再看 Les Combes 的攻防。斯帕长圈让不同区域天气可能不同，也让安全车和进站窗口更难判断。

## 为什么这条赛道容易 / 不容易超车？

斯帕有长直道和重刹车点，具备超车条件。但高速弯和下压力取舍会让赛车表现分区明显：低阻设定直道快，高下压力设定弯中稳，攻防并不只看一段。

## 排位赛看点

排位看低阻与下压力取舍、长圈 traffic、以及天气变化。斯帕一圈很长，某一段小雨或黄旗都可能让快圈失去代表性。

## 正赛看点

正赛重点看天气、安全车和不同下压力设定的攻防。长直道适合追击，但轮胎温度和高速弯稳定性也会限制后车持续跟近。
